# 贡献者认可系统

## 目标

使用 all-contributors 规范在 README 中显示贡献者，支持教程、代码、翻译等多种贡献类型的认可。

## 验收标准

- [ ] all-contributors bot 已在仓库中配置
- [ ] README 中显示贡献者表格
- [ ] 支持多种贡献类型（教程、代码、翻译、review 等）
- [ ] 贡献者添加流程已文档化

## 实施说明

1. 在仓库中安装 [all-contributors bot](https://allcontributors.org/)
2. 创建 `.all-contributorsrc` 配置文件，定义贡献类型：
   - `code` - 代码贡献
   - `doc` - 教程和文档编写
   - `translation` - 翻译工作
   - `review` - 内容审核
   - `ideas` - 提供想法和建议
   - `maintenance` - 项目维护
3. 在 README.md 中添加 `<!-- ALL-CONTRIBUTORS-LIST:START -->` 占位
4. 使用 `@all-contributors please add @username for contribution_type` 命令添加贡献者

## 涉及文件

- `README.md`
- `.all-contributorsrc`
