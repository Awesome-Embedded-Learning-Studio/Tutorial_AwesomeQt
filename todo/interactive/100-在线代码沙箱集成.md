# 在线代码沙箱集成

## 目标

集成 Compiler Explorer/Godbolt iframe，在教程页面嵌入交互式代码示例，让读者可以直接在浏览器中编译和运行代码片段。

## 验收标准

- [ ] 代码块旁有 "Run" 按钮
- [ ] 点击按钮弹出 iframe 可编辑运行代码
- [ ] 至少在专家层源码分析教程中使用
- [ ] 支持多种编译器选择（gcc, clang, MSVC）
- [ ] 集成不影响页面加载性能

## 实施说明

1. **集成方案**：
   - 使用 Compiler Explorer (Godbolt) 的 iframe 嵌入功能
   - 创建自定义 JavaScript 按钮组件
   - 代码块通过 URL 参数传递给 Godbolt

2. **实现步骤**：
   - 编写 `javascripts/code-runner.js`，为指定代码块添加 "Run" 按钮
   - 使用 Godbolt 的 URL API 传递代码：
     ```
     https://godbolt.org/api/shortlink#code=...
     ```
   - 点击按钮时弹出 iframe 模态框

3. **适用范围**：
   - 专家层 Qt 源码分析教程（不需要 GUI）
   - 算法和数据结构示例
   - C++ 语言特性演示

4. **注意事项**：
   - Qt GUI 程序无法在 Godbolt 运行，仅适用于控制台示例
   - 需要考虑网络延迟和离线场景
   - iframe 尺寸需要响应式适配

## 涉及文件

- `javascripts/code-runner.js`
- `mkdocs.yml`（引入脚本）
- 相关教程 Markdown 文件
