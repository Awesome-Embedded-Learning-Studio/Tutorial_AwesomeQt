# 练习题交互系统

## 目标

增强教程中的练习题，支持代码填空自动验证，提升学习体验的互动性和即时反馈能力。

## 验收标准

- [ ] 练习题可交互操作
- [ ] 代码填空题可自动验证答案
- [ ] 支持提示功能（逐级提示）
- [ ] 验证结果有视觉反馈（正确/错误提示）
- [ ] 系统在 Material 明暗模式下均可正常使用

## 实施说明

1. **练习题类型**：
   - **选择题**：单选/多选，即时反馈
   - **代码填空**：输入缺失代码片段，自动验证
   - **代码排序**：拖拽代码行到正确顺序
   - **简答题**：自由输入，提供参考答案对比

2. **技术实现**：
   - 创建 `javascripts/exercises.js` 练习题引擎
   - 练习题数据以 JSON 格式嵌入教程页面
   - 使用 `<div class="exercise">` 占位元素
   - JavaScript 渲染交互组件并处理验证逻辑

3. **代码填空验证**：
   - 定义标准答案和可接受的变体
   - 支持正则表达式匹配
   - 忽略空白差异
   - 提供逐级提示（最多 3 级）

4. **数据格式**：
   ```json
   {
     "type": "fill-blank",
     "question": "完成信号槽连接",
     "template": "QObject::connect(btn, &QPushButton::_____, this, &MainWindow::onClicked);",
     "answer": ["clicked"],
     "hints": ["这是 QPushButton 的点击信号", "信号名称与用户操作相关"]
   }
   ```

## 涉及文件

- `javascripts/exercises.js`
- `javascripts/exercises.css`
- `mkdocs.yml`（引入脚本和样式）
- 相关教程 Markdown 文件
