# 可视化架构图表集

## 目标

创建 Qt 核心架构的可视化图表集（对象树、信号槽、事件系统、MVC、嵌入式构建流水线），使用 Mermaid 绘制并在对应教程中引用。

## 验收标准

- [ ] 至少 5 张核心架构图已创建
- [ ] 所有图表使用 Mermaid 绘制
- [ ] 在对应教程中引用这些图表
- [ ] 图表在明暗模式下均可正常显示
- [ ] 图表可复用且易于维护

## 实施说明

1. **图表清单**：
   - **Qt 对象树**：展示 QObject 父子关系、内存管理机制
   - **信号槽连接**：展示信号发射、槽函数调用的序列流程
   - **事件系统**：展示事件循环、事件过滤器、事件处理链路
   - **MVC 架构**：展示 Model-View-Controller 在 Qt 中的实现
   - **嵌入式构建流水线**：展示交叉编译、部署到目标板的全流程

2. **实现方式**：
   - 每张图表创建为独立的 Mermaid 代码片段
   - 使用 `082-Mermaid图表增强` 任务中定义的自定义主题
   - 图表文件存放在 `document/diagrams/` 目录下
   - 在教程中通过 `{% include %}` 或直接嵌入引用

3. **质量要求**：
   - 图表标注清晰，关键节点有中文说明
   - 颜色编码一致（如：输入用蓝色、输出用绿色、错误用红色）
   - 图表尺寸适中，不过于复杂

## 涉及文件

- `document/diagrams/` 目录下的各 Mermaid 文件
- 相关教程 Markdown 文件
