# GIF 演示自动生成

## 目标

创建脚本自动录制 Widget/App 运行效果并生成 demo.gif，配合 083-GIF演示规范 实现演示素材的自动化生成。

## 验收标准

- [ ] scripts/generate_demo_gif.sh 可自动录制指定项目
- [ ] 输出标准尺寸 GIF（宽度 800px，5-15 秒）
- [ ] 支持 Linux 桌面环境录制
- [ ] GIF 文件大小不超过 2MB
- [ ] 支持批量处理多个项目

## 实施说明

1. **脚本功能**：
   ```bash
   ./scripts/generate_demo_gif.sh --project codes/widgets/01_hello_button
   ```
2. **录制方案**：
   - 使用 `ffmpeg` + X11 截屏录制
   - 或使用 `byzanz-record` 直接输出 GIF
   - 或使用 `xdotool` 模拟用户交互 + 录屏

3. **处理流程**：
   - 编译目标项目
   - 启动程序并录制指定时长
   - 使用 `ffmpeg` 转换为 GIF 并优化
   - 输出到项目 `assets/` 目录

4. **GIF 优化**：
   ```bash
   ffmpeg -i recording.mp4 \
     -vf "fps=15,scale=800:-1:flags=lanczos" \
     -loop 0 output.gif
   ```

5. **CI 集成**（可选）：
   - 在 GitHub Actions 中使用虚拟显示器 (Xvfb)
   - 自动录制并上传 GIF
