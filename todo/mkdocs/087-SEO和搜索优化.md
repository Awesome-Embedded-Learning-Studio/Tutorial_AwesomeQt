# SEO 和搜索优化

## 目标

优化站点搜索体验（中文分词已配置 jieba）、添加 sitemap、为每页添加 meta 描述，提升站点的可发现性和搜索质量。

## 验收标准

- [ ] jieba 中文分词正常工作
- [ ] sitemap.xml 在构建后自动生成
- [ ] 每页有 meta description 标签
- [ ] 搜索结果中文分词准确
- [ ] robots.txt 配置正确

## 实施说明

1. **搜索优化**：
   - 确认 `mkdocs.yml` 中 jieba 分词配置正确
   - 测试中文搜索结果的准确性
   - 调整搜索索引策略（如需要）

2. **Sitemap**：
   - 启用 `mkdocs.yml` 中的 sitemap 生成
   - 配置 site_url
   - 验证 sitemap.xml 内容

3. **Meta 描述**：
   - 为每个教程页面添加 meta description
   - 在 front matter 或 `mkdocs.yml` 中配置
   - 确保描述长度适中（120-160 字符）

4. **robots.txt**：
   - 创建 `docs/robots.txt`
   - 配置允许/禁止爬取的路径

## 涉及文件

- `mkdocs.yml`
- `docs/robots.txt`（如需要）
- 各教程 Markdown 文件的 front matter
