# 标签系统配置

## 目标

添加 Material tags 插件，支持按 `#beginner` `#signal-slot` `#embedded` 等标签交叉检索教程内容。

## 验收标准

- [ ] mkdocs.yml 添加 tags 插件配置
- [ ] 创建 document/tutorials/tags.md 标签索引页
- [ ] 所有教程文件添加 tags 元数据字段
- [ ] 标签页面可正常按标签筛选和导航

## 实施说明

1. 在 `mkdocs.yml` 中启用 `tags` 插件，配置标签索引页路径
2. 定义一套标准标签词汇表（难度等级、技术主题、平台等维度）
3. 为每个教程 Markdown 文件添加 `tags` front matter 元数据
4. 创建 `document/tutorials/tags.md` 作为标签聚合索引页

## 涉及文件

- `mkdocs.yml`
- `document/tutorials/tags.md`
