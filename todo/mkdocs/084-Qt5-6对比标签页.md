# Qt 5/6 对比标签页

## 目标

使用 pymdownx.tabbed 在教程中展示 Qt 5/6 双版本写法对比，让读者能够同时看到两个版本的差异。

## 验收标准

- [ ] 定义标准代码块标签页模板
- [ ] 在信号槽相关教程中使用 Qt 5/6 对比标签页
- [ ] 在构建系统教程中使用 Qt 5/6 对比标签页
- [ ] 至少在 3 个关键教程中使用对比标签页
- [ ] 标签页样式在明暗模式下正常显示

## 实施说明

1. 确保 `mkdocs.yml` 已启用 `pymdownx.tabbed` 扩展
2. 定义标准模板格式：
   ```markdown
   === "Qt 5"

       ```cpp
       // Qt 5 code
       ```

   === "Qt 6"

       ```cpp
       // Qt 6 code
       ```
   ```
3. 优先在以下教程中应用：
   - 信号槽新语法（connect 语法变化）
   - 构建系统（qmake vs CMake）
   - 容器类变化（QList, QHash 等）

## 涉及文件

- `mkdocs.yml`
- 相关教程 Markdown 文件
