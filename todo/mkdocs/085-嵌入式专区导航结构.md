# 嵌入式专区导航结构

## 目标

设计嵌入式教程的导航结构，按板级分类，支持快速定位，为嵌入式教程内容提供清晰的层次化组织。

## 验收标准

- [ ] document/tutorials/embedded/.pages 配置完成
- [ ] 导航层级清晰：概览 -> 板级 -> 主题
- [ ] 嵌入式专区在主导航栏中可见
- [ ] 每个板级分类有独立的索引页
- [ ] 支持按开发板（如 Raspberry Pi、STM32 等）筛选

## 实施说明

1. 在 `document/tutorials/embedded/` 目录下创建 `.pages` 文件配置导航
2. 设计目录结构：
   ```
   embedded/
     index.md          (概览页)
     .pages             (导航配置)
     raspberry-pi/
       .pages
       index.md
       ...
     stm32/
       .pages
       index.md
       ...
   ```
3. 配置导航标题和排序
4. 在上级 `.pages` 中引用嵌入式专区

## 涉及文件

- `document/tutorials/embedded/.pages`
- `document/tutorials/embedded/index.md`
