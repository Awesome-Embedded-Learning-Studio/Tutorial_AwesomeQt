# GIF 演示规范

## 目标

定义每个 Widget/App 项目的 demo.gif 规范和自动生成流程，使教程附带直观的运行效果展示。

## 验收标准

- [ ] 每个 Widget 项目包含 demo.gif 文件
- [ ] 在目录页（catalog）引用对应 demo.gif
- [ ] 定义 GIF 尺寸规范（建议宽度）
- [ ] 定义 GIF 时长规范（建议 5-15 秒）
- [ ] 编写 GIF 录制规范文档

## 实施说明

1. 定义 GIF 规范：
   - 尺寸：宽度不超过 800px
   - 时长：5-15 秒，循环播放
   - 帧率：15fps
   - 文件大小：不超过 2MB
2. 在每个 Widget/App 项目目录下创建 `assets/demo.gif`
3. 在目录页 Markdown 中通过 `![demo](assets/demo.gif)` 引用
4. 可配合 102-GIF演示自动生成 任务实现自动化

## 涉及文件

- 各 Widget/App 项目目录下的 `assets/demo.gif`
- 目录页文件
