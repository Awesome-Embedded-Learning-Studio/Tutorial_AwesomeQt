# 进度追踪小部件

## 目标

首页添加各模块完成进度条，基于 JSON 数据自动更新，让访问者一目了然地了解教程的覆盖情况。

## 验收标准

- [ ] extra_javascript 添加 progress.js 脚本
- [ ] 首页显示总进度条和各模块独立进度条
- [ ] 进度数据来源于 progress.json 文件
- [ ] progress.json 数据结构与现有教程模块对应
- [ ] 进度条在 Material 明暗模式下均可正常显示

## 实施说明

1. 创建 `progress.json`，定义各模块的教程总数和已完成数
2. 编写 `javascripts/progress.js`，读取 JSON 并渲染进度条 HTML
3. 在 `mkdocs.yml` 的 `extra_javascript` 中引入该脚本
4. 在首页 Markdown 中预留进度条占位元素

## 涉及文件

- `mkdocs.yml`
- `javascripts/progress.js`
- `progress.json`
