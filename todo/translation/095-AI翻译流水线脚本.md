# AI 翻译流水线脚本

## 目标

创建 scripts/translate.py，使用 Claude API 将中文教程自动翻译为英文，支持批量翻译和技术术语对照。

## 验收标准

- [ ] 支持 --source 参数指定源文件/目录
- [ ] 支持 --lang 参数指定目标语言
- [ ] 翻译结果保持 Markdown 格式完整
- [ ] 内置技术术语对照表（来自 098-中英术语对照表）
- [ ] 支持批量翻译（目录级别）
- [ ] 翻译结果输出到对应的语言目录

## 实施说明

1. **脚本设计**：
   ```bash
   python scripts/translate.py --source document/tutorials/beginner/ --lang en
   ```
2. **核心功能**：
   - 读取中文 Markdown 源文件
   - 调用 Claude API 进行翻译
   - 使用 system prompt 约束翻译风格
   - 参照术语对照表保持一致性
   - 输出英文 Markdown 文件

3. **API 调用策略**：
   - 按 Markdown 文件为单位翻译
   - 保留 front matter 不翻译
   - 保留代码块不翻译，翻译代码注释
   - 保留 Mermaid 图表不翻译，翻译图表中的中文标签

4. **错误处理**：
   - API 调用失败重试
   - 文件编码处理
   - 速率限制处理
