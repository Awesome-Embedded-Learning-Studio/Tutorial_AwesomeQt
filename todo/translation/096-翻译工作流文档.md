# 翻译工作流文档

## 目标

编写翻译工作流说明文档，包含术语对照、翻译质量标准、审核流程，确保翻译工作的规范性和一致性。

- [ ] 翻译工作流程已完整记录

## 实施说明

1. **工作流程**：
   - 使用 `scripts/translate.py` 进行初译
   - 人工审核和修正
   - 提交翻译 PR
   - 审核者 review 并合并

2. **术语对照规则**：
   - 引用 098-中英术语对照表
   - 说明专有名词的翻译策略
   - 定义不翻译的术语列表

3. **质量标准**：
   - 准确性：技术内容无误差
   - 流畅性：英文表达自然
   - 一致性：术语翻译统一
   - 格式：Markdown 格式完整保留

4. **审核流程**：
   - 自审检查清单
   - PR 审核重点
   - 审核者职责

5. **常见问题**：
   - Qt 专有术语处理
   - 代码注释翻译策略
   - 中英文混排规范
