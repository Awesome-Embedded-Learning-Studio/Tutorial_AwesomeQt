# 双语站点配置

## 目标

配置 MkDocs i18n 插件或手动方案，支持中英文切换，使站点能够同时提供中文和英文版本的教程。

## 验收标准

- [ ] 站点支持语言切换（中/英）
- [ ] URL 结构合理（/en/ /zh/）
- [ ] 语言切换按钮在导航栏可见
- [ ] 默认语言为中文
- [ ] 搜索支持多语言

## 实施说明

1. **方案选择**：
   - 优先考虑 `mkdocs-static-i18n` 插件
   - 备选方案：手动构建多版本

2. **使用 mkdocs-static-i18n**：
   ```yaml
   plugins:
     - i18n:
         default: zh
         languages:
           zh: 中文
           en: English
   ```
3. **文件组织**：
   ```
   document/tutorials/
     beginner/
       index.md          (中文默认)
       index.en.md        (英文版本)
   ```
4. **URL 结构**：
   - 中文：`/tutorials/beginner/`
   - 英文：`/en/tutorials/beginner/`

5. **导航本地化**：
   - 翻译导航标题
   - 翻译页脚和标签

## 涉及文件

- `mkdocs.yml`
- 各教程文件的 `.en.md` 版本
