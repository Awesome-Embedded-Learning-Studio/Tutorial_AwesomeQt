# 工业级项目模板目录

本目录列出所有规划的工业级 Qt 项目模板。每个模板是可直接复用的完整项目骨架，包含典型工业场景的 UI 布局、数据绑定、通信协议等。

## 来源说明

| 代码 | 说明 |
|------|------|
| NEW | 原创工业模板 |

## 优先级定义

| 级别 | 含义 |
|------|------|
| P0 | 核心工业场景，高频需求 |
| P1 | 常见工业应用 |
| P2 | 特定领域，低频但高价值 |
