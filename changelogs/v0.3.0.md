## 站点改进

### 旗舰化基建（PR#13）

- **单一真相源**：抽 `shared.ts` 统一 dev / build 双配置，根治历史 `defineConfig` 漂移。
- **CI 质量门**（四道）：
  - `build-check` — VitePress 构建验证
  - `lint` — markdownlint 强制拦截
  - `content-quality` — 内部链接可达性 + 专家层源码引用验真
  - `build-examples` — 275 个示例工程编译门（install-qt-action 装全套 Qt6 模块）
- **SEO 基础**：sitemap.xml + robots.txt + favicon 修复，供搜索引擎收录。
- **站点体验**：代码折叠（>30 行）、字号五档切换、快捷键、页脚版本号、mermaid 图表渲染、首页 3D coverflow 轮播。
- **License NOTICE**：代码 MIT / 文档 CC BY-SA 4.0 / Qt 源码引用按 Qt Company 许可，分档授权写清。
- **贡献者入口**：CONTRIBUTING / AGENTS / Issue & PR 模板。
- third_party submodule 钉到 release tag（benchmark v1.9.5 / googletest v1.17.0）。

### 站点样式大更新（PR#17）

- **Qt 绿身份重塑**：亮 / 暗双主题统一品牌色。
- **可拖拽侧栏**：左右栏宽度可调，localStorage 持久化。
- **阅读进度条** + **长代码折叠** + **字号五档切换**。

### 宣传门面统一

- README 加在线站点入口、实例库双表（widget 13 + app 7）、进度数字纠正、markdownlint 清零。
- 首页 tagline 改「聚焦 C++/QtWidgets · 三层递进 · 学→做闭合」。
- 专家层导航页按 102 篇重写。
