## 站点改进

### VitePress 迁移

- 从 MkDocs Material 迁移至 VitePress
- 自动侧边栏生成（`buildSidebar()` 扫描目录结构）
- 导航栏新增「进阶教程」下拉菜单
- 首页 Roadmap 更新：进阶层状态改为「编写中 · 22 篇已完成」

### 卡片布局

- 新增 `CardGrid` / `CardLink` Vue 组件
- 全部 12 个 index.md 从纯文本列表升级为响应式卡片网格
- 卡片支持序号标签、标题、描述，hover 动效，暗色模式适配

### Metadata 标准化

- 全站 192 个 markdown 文件统一添加 frontmatter（`title` + `description`）
- 侧边栏标题精简：如 "现代Qt开发教程（新手篇）1.1——QObject 与元对象系统" → "1.1 QObject 与元对象系统"

### 安全修复

- C++ 模板转义插件从宽泛的 PascalCase 正则匹配改为显式 Vue 组件白名单，避免 `<QWidget>` 等被误放行
