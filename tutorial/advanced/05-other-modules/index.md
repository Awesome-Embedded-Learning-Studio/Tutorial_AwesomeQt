# 05 · 其他扩展模块（进阶）

## 模块概述

本模块覆盖 Qt 生态中除 QtBase/QtGui/QtWidgets/QtNetwork/QML 以外的 25 个扩展模块，每篇在入门基础上深入高级 API、协议细节、性能优化与工程实践。

## 章节列表

- **01 QtSql** — 事务控制、连接池、轻量 ORM 封装
- **02 QSqlRelationalTableModel** — 关联表视图与外键联动
- **03 QtCharts / QtGraphs** — 实时数据更新、自定义 Axis
- **04 媒体播放** — 播放列表、元数据、字幕
- **05 摄像头** — 视频录制、帧处理、滤镜
- **06 QtSvg** — 动态修改 SVG 元素属性
- **07 打印支持** — 复杂报表生成与多页面布局
- **08 Modbus** — RTU/TCP 切换、寄存器映射表管理
- **09 MQTT** — QoS 1/2、遗嘱消息、TLS 加密
- **10 蓝牙** — GATT Profile 读写 Characteristic
- **11 NFC** — NDEF 记录类型详解与写入标签
- **12 状态机** — 层次状态机、历史状态、并行状态
- **13 SCXML** — 数据模型与延迟事件
- **14 Qt3D** — 自定义 Material、Framegraph 配置
- **15 QtQuick3D** — PBR 材质、环境光遮蔽、阴影
- **16 QtQuick3D Physics** — 关节约束、力与冲量、射线检测
- **17 QtPdf** — 文本搜索、选中复制、书签导航
- **18 QtHttpServer** — 中间件链、静态文件服务、身份验证
- **19 QtWebSockets 服务端** — 房间广播、消息队列
- **20 QtWebChannel** — 自定义传输层（非 WebEngine）
- **21 QtWebEngine** — 自定义 URL Scheme、安全策略
- **22 QtRemoteObjects** — 自定义序列化与网络传输
- **23 QtSpatialAudio** — 混响、距离衰减、头部追踪
- **24 QtTextToSpeech** — SSML 标记语言与语音合成控制
- **25 Qt5Compat** — 批量迁移策略与自动化检测工具
